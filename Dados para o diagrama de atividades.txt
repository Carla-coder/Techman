
Descrição geral do fluxo:

Início: O processo começa com o usuário tentando fazer login.

Verificação de credenciais: O sistema verifica se as credenciais (usuário e senha) correspondem 
a um administrador.

Login bem-sucedido: Se as credenciais forem válidas, o usuário é direcionado para a tela de gestão 
de equipamentos.

Inclusão de novo equipamento: O administrador insere os dados do novo equipamento.

Validação dos dados: O sistema valida se todos os campos obrigatórios foram preenchidos 
corretamente.

Confirmação de registro: Após a validação, o sistema registra o equipamento e notifica o 
administrador.

Fim: O processo termina com a confirmação de que o equipamento foi adicionado com sucesso.

Atividades no diagrama:

Usuário insere credenciais (início)

O administrador insere a senha.

Sistema verifica credenciais

O sistema compara as credenciais com as armazenadas no banco de dados.

Login autorizado?

Se o login for bem-sucedido (credenciais corretas), o sistema redireciona o usuário para 
a tela de gestão de equipamentos.

Se o login falhar (credenciais incorretas), o sistema retorna uma mensagem de erro.

Administrador acessa a tela de gestão de equipamentos

O administrador acessa a tela onde pode visualizar, editar e adicionar equipamentos.

Administrador seleciona "Adicionar Equipamento"

O administrador opta por adicionar um novo equipamento ao sistema.

Administrador insere os dados do equipamento

O administrador preenche o formulário com informações como nome do equipamento, modelo, 
número de série, entre outros.

Sistema valida os dados

O sistema verifica se todos os campos obrigatórios foram preenchidos corretamente e se as 
informações estão no formato esperado.

Dados válidos?

Se os dados estiverem corretos, o processo segue para o registro.

Se houver erro (como campos vazios ou informações incorretas), o sistema retorna uma mensagem 
de erro solicitando correção.

Sistema registra o equipamento

O sistema salva os dados do novo equipamento no banco de dados.

Exibe confirmação

O sistema exibe uma mensagem confirmando que o equipamento foi registrado com sucesso.

Fim do processo

Fluxo de exceção:

Se o login falhar, o sistema exibe uma mensagem de erro e solicita a reinserção das credenciais.

Se os dados do equipamento forem inválidos, o sistema retorna um erro e o administrador deve 
corrigir as informações. 

***************************************************************************************************
1. Login do Usuário Administrador:

Início: O administrador acessa a tela de login.

Entrada de Credenciais: O administrador insere o nome de usuário e senha.

Validação de Credenciais: O sistema verifica se as credenciais são válidas.

Se inválido: O sistema exibe uma mensagem de erro e solicita nova tentativa.

Se válido: O sistema permite o acesso ao painel administrativo.

2. Registro de Novo Equipamento:

Acesso ao Menu de Equipamentos: O administrador navega até a seção de gerenciamento de equipamentos.

Formulário de Inclusão de Equipamento: O administrador preenche o formulário com as informações do 
equipamento 
(ex.: nome, descrição, número de série).

Envio do Formulário: O administrador envia os dados.

Validação de Dados: O sistema valida as informações inseridas (ex.: campos obrigatórios, formato dos dados).

Se inválido: O sistema exibe uma mensagem de erro e solicita correção.

Se válido: O sistema registra o equipamento na base de dados.

Confirmação de Registro: O sistema exibe uma mensagem de sucesso.

3. Fim: O administrador finaliza a operação.

***************************************************************************************************
As formas principais que você usará são:
Ellipse (Elipse): Para o Início e Fim das atividades.
Rectangle (Retângulo): Para cada ação/atividade que o usuário realiza.
Diamond (Losango): Para decisões ou verificações no fluxo.
Arrow (Setas): Para conectar os elementos e indicar o fluxo.

Montar o Fluxo do Login do Administrador

Início:

Arraste uma elipse para o espaço de trabalho e insira o texto "Início".

Ação: Tela de Login:

Adicione um retângulo logo abaixo do "Início" e nomeie como "Administrador acessa a tela de login".

Conecte o "Início" ao retângulo com uma seta (Arraste da borda de um elemento para outro).

Ação: Entrada de Credenciais:

Adicione outro retângulo e insira "Administrador insere nome de usuário e senha".

Conecte o retângulo anterior a este novo retângulo com uma seta.

Decisão: Validação das Credenciais:

Arraste um losango e adicione o texto "Credenciais válidas?".

Conecte o retângulo de "Inserir credenciais" ao losango com uma seta.

Decisão Negativa (Credenciais Inválidas):

Adicione um retângulo à esquerda do losango e nomeie como "Exibir erro: credenciais inválidas".

Conecte o losango a este retângulo com uma seta marcada como "Não".

Adicione uma seta que volta do retângulo "Erro" para o retângulo de "Inserir Credenciais".

Decisão Positiva (Credenciais Válidas):

Adicione um retângulo abaixo do losango e insira "Permitir acesso ao painel administrativo".

Conecte o losango a esse retângulo com uma seta marcada como "Sim".

Fluxo de Inclusão de Equipamento

Ação: Acesso ao Menu de Equipamentos:

Adicione um retângulo logo abaixo do "Permitir acesso ao painel" e nomeie "Administrador acessa menu de equipamentos".

Conecte o último retângulo do login com essa nova ação com uma seta.

Ação: Formulário de Inclusão de Equipamento:

Adicione outro retângulo e insira "Preenche formulário de novo equipamento".

Conecte o retângulo anterior a este.

Decisão: Validação dos Dados:

Adicione um losango e insira "Dados válidos?".

Conecte o retângulo do formulário ao losango com uma seta.

Decisão Negativa (Dados Inválidos):

Adicione um retângulo à esquerda do losango e nomeie "Exibir erro: Dados inválidos".

Conecte o losango a esse retângulo com uma seta marcada como "Não".

Adicione uma seta que volta do retângulo "Erro" para o retângulo de "Preencher Formulário".

Decisão Positiva (Dados Válidos):

Adicione um retângulo abaixo do losango e nomeie "Registrar equipamento no sistema".

Conecte o losango a esse retângulo com uma seta marcada como "Sim".

Ação: Confirmação de Registro:

Adicione um retângulo abaixo do anterior e nomeie "Exibir mensagem de sucesso".

Conecte o retângulo de "Registrar equipamento" com esse novo retângulo.

Fim:

Adicione uma elipse com o texto "Fim".

Conecte a última ação ("Exibir mensagem de sucesso") ao "Fim" com uma seta.

Ajustes Finais

Após adicionar todos os elementos, alinhe as formas de maneira organizada.

Use setas para conectar todas as etapas, garantindo que o fluxo siga o caminho correto.

Ajuste o tamanho do texto e formas, conforme necessário para tornar o diagrama claro e legível.

Agora, seu diagrama de atividades deve representar os dois processos principais: Login do Administrador e Registro de Equipamento!
